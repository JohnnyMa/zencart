Installation notes:
-------------------

1. Backup all the files and database.
2. Upload the Template package (zip file) to the root directory of your Zen Cart installation (for example: www.yourdomain.com/yourzencartstore/)
3. Unpack the Template package.
4. Log in to the admin area.
5. Go to the Tools -> Template Selection.
6. Click on the 'Classic Contemporary Green' template name.
7. Select 'BlackPureFree' value for the selectbox on the right and click on the 'Update' button.
8. Go to the Tools -> Layout Boxes Controller.
9. Click on the 'Reset' button on the bottom.
10. Enjoy.

You can configure Template via 'Tools->Layout Boxes Controller' and 'Configuration->Layout settings' sections in the admin area.


HOME PAGE BANNERS SETTING UP
----------------------------

You can manage the home page banners using the admin area.

To add one of them you should:
              
1. Log in to the admin area.
2. Go to the 'Tools -> Banner Manager'.
3. Click on the 'New banner' button.
4. Fill the 'Banner Title' field.
5. Fill the 'Banner URL' optional field.
6. Enter a new banner group in the 'Banner Group' field (for example 'Home-Banner').
7. Upload an image from your local computer using 'Image' field of this section.
8. Click on the 'Insert' button at the bottom of the page.
9. New banner will be created.

To appear the home page banner you should:

1. Go to the 'Configuration->Layout settings'.
2. Click on the 'Banner Display Groups - Header Position 1' or 'Banner Display Groups - Header Position 2' title.
3. Enter the banner group name you created ('Home-Banner' or yours) in the field on the right and click on the 'Update' button.
4. Banner will appear on the home page.

You can disable 'Open New Window' property of your banner by clicking on the orange icon in the 'New Window' column on the 'Tools -> Banner Manager' page.
